# Vim的基本用法与进阶

vim作为开源默认的编辑器，为越来越多的人了解和熟悉。vim是vi的变种之一，集成了vi的迅捷编辑方式，在功能上也逼原始的vi 强大很多

## vim基本用法

### 1.命令模式

```
h或←方向键====>  # 光标向左移动一个字符
j或↓方向键====>  # 光标向下移动一个字符
k或↑方向键====>  # 光标想上移动一个字符
i或→方向键====>  # 光标向右移动一个字符
n<space>   # 按下数字键再按空格，光标向→移动n个字符
0<HOME>    # 移动到这一行的第一个字符
$<END>     # 移动到这一行的最后一个字符
G          # 光标移动到文件的最后一行
nG         # 光标移动到文件的第n行
n<Enter>   # 光标向下移动n行
/word      # 在光标之后查找一个名为word的字符串
?word      # 在光标之前查找一个名为word的字符串
:n1,n2 w [filename]     # 将n1到n2的内容存储成filename
:n1,n2s/word1/word2/g   # 在第n1与n2行之间查找word1这个字符串，并替换为word2
:1,$s/word1/word2/g     # 从第一行到末行查找替换
:1,$s/word1/word2/gc    # 从第一行到末行查找替换，需用户确认
:w !sudo tee %   		# 强制保存并退出
```
### 2.一般模式

```
x,X        # x为向后删除一个字符，X为向前删除一个字符
ndd        # 删除光标所在列的向下n行
nyy        # 复制光标所在向下的n行
p,P        # p为复制的数据粘贴在光标下一列，P为粘贴在光标的上一列
u          # 恢复前一个动作
a,A        # 添加，在当前的光标所在处下一个字符开始输入，已存在的字符会自动后退
o,O        # 插入新的一行，从光标所在行的下一行行首开始输入字符
r,R        # 替换，r会替换光标所在的那一个字符；R会一致替换光标所指的文字，除非你按ESC
Esc        # 退出编辑模式，回到一般模式
```
